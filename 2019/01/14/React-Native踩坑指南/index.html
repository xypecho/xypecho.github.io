React Native踩坑指南

1.初次编译运行 React Native 应用时的坑

首先按照react native的中文网上的说明，一步步搭建环境这边是没什么坑的，大部分都坑在react-native run-android真机运行这个上面。基本都是报错 :app:installDebug FAILED,这里的答案足够解决很多问题，但是！！！如果你的尝试了google到的所有答案，都还没解决的话可以试试升级下USB驱动…我就是因为这个问题被坑了整整三天…

2.项目已经安装到手机了，与电脑断开连接后如何再次连接调试

首先把你的设备通过 USB 数据线连接到电脑上，并开启 USB 调试。
运行项目react-native start,
然后在项目的根目录打开cmd，运行adb reverse tcp:8081 tcp:8081

3.真机调试时，每次更新代码都要摇一摇手机？

项目根目录执行adb shell input keyevent 82,解放你的双手
