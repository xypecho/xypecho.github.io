css实现保留返回数据的回车符

对于一般的从后端拿的数据中包含换行符的，要么是使用正则来匹配替换，要么是用’pre’这个标签来实现.
但是在用vue+element项目中的表格里面时字段都是用props传进去的，用formatter格式也很复杂，这时就需要一个css的黑科技属性了

white-space: pre (空白会被浏览器保留。其行为方式类似 HTML 中的 pre 标签)
