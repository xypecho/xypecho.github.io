icomoonжҳҜд»Җд№Ҳпјҹ

жү“ејҖIcoMoonзҪ‘з«ҷеҫҖеҫҖдјҡз»ҷдәәдёҖз§Қй”ҷи§үпјҢиҜҜи®Өдёәи·ҹдёҠйқўзҪ—еҲ—зҡ„еӯ—дҪ“зҪ‘з«ҷдёҖж ·пјҢе°ұжҳҜеұ•зӨәгҖӮ
е…¶е®һйқһд№ҹпјҢIcoMoonзҡ„ж°ҙиҰҒж·ұеӨҡдәҶгҖӮ
иҝҷд№ҲиҜҙеҗ§пјҡеҒҮи®ҫж•ҙдёӘдёҠжө·жў…еӣӯең°еҢәжңү100е®¶еҚ–иЎЈжңҚзҡ„е°Ҹеә—пјҢе…¶дёӯжңү99е®¶е°ұжҳҜжӢҝиҙ§з„¶еҗҺеҚ–иЎЈжңҚз»ҷйЎҫе®ўпјҲиҝҷе°ұжҳҜеӨ§жҗңзҪ—зҡ„icon fontд»¬пјүпјӣдҪҶжҳҜпјҢе…¶дёӯжңүдёҖе®¶еә—дёҚд»…еҚ–иЎЈжңҚпјҢиҝҳеҸҜд»Ҙж №жҚ®з”ЁжҲ·йңҖжұӮе®ҡеҲ¶иЎЈжңҚпјҢз”ҡиҮіиҝҳеҸҜд»ҘжҠҠеҲ«е®¶еә—зҡ„иЎЈжңҚжҢүз…§йЎҫе®ўиҰҒжұӮиҝӣиЎҢж”№еҲ¶пјҢдё”йҖӮз”ЁдәҺй«ҳзҹ®иғ–зҳҰпјҢеҸӨд»ҠдёӯеӨ–з»ҷзұ»дәәзҫӨпјҢиҖҢдё”иҝҳжңүз©ҝиЎЈиҜҙжҳҺпјҢжЁЎзү№иҜ•з©ҝд»ҘеҸҠжү“еҢ…йҖҒиҙ§дёҠй—ЁжңҚеҠЎпјҲиҝҷе°ұжҳҜжң¬ж–Үзҡ„IcoMoonпјүпјҢжңҖжңҖе…ій”®зҡ„жҳҜпјҢиҝҷе®¶еә—жҳҜе…Қиҙ№зҡ„гҖӮ
дҪ“зҺ°еңЁIcoMoonдёҠе°ұжҳҜпјҡжҸҗдҫӣ600+еӯ—з¬ҰпјҢжӮЁеҸҜд»Ҙж №жҚ®иҮӘе·ұйңҖжұӮе®ҡеҲ¶пјҲеҰӮе°ұйҖүдёӨдёӘпјүпјӣеҸҜд»ҘеҜје…Ҙе…¶д»–еӯ—дҪ“пјҢд№ҹиҝӣиЎҢзү№еҲ«е®ҡеҲ¶пјҲзұ»дјјfontforgeеҠҹиғҪпјүпјӣе®ҡеҲ¶еӯ—дҪ“жҸҗдҫӣжү“еҢ…еҜјеҮәеҠҹиғҪпјҲзңҒеҺ»дәҶеӯ—дҪ“иҪ¬жҚўпјүпјҢе…је®№IE6+пјҢзҺ°д»ЈжөҸи§ҲеҷЁд»ҘеҸҠеҗ„зұ»жүӢжңәи®ҫеӨҮпјҢдё”жңүdemoе®һдҫӢпјҢе№¶еҜ№еӯ—з¬ҰиҝӣиЎҢдәҶHTMLиҪ¬еҢ–гҖӮ
еҸҜи°“еӣҫж Үеӯ—дҪ“дёҖжқЎйҫҷжңҚеҠЎпјҢеҸҲз§°вҖңеӣҫж Үи§ЈеҶіж–№жЎҲвҖқпјҒ

дҪҝз”Ёж–№жі•

1гҖҒжү“ејҖicomoonзҡ„е®ҳзҪ‘,йҖүжӢ©иҮӘе·ұйңҖиҰҒзҡ„еӣҫж ҮпјҢеҪ“з„¶дҪ д№ҹеҸҜд»ҘзӮ№еҮ»е·ҰдёҠи§’зҡ„вҖҷimport iconsвҖҷеҜје…ҘиҮӘе·ұзҡ„svgж јејҸзҡ„еӣҫж ҮгҖӮ
йҖүжӢ©е®ҢйңҖиҰҒзҡ„еӣҫзүҮеҗҺзӮ№еҮ»дёӢйқўзҡ„creat font
з„¶еҗҺзӮ№еҮ»downloadе°ұдјҡжҠҠеӯ—дҪ“еӣҫж ҮдёӢиҪҪеҲ°жң¬ең°гҖӮ
2гҖҒи§ЈеҺӢеҺӢзј©еҢ…иҺ·еҫ—д»ҘдёӢж–Үд»¶
3гҖҒеңЁиҮӘе·ұзҡ„vueйЎ№зӣ®srcж–Үд»¶дёӢеҲӣе»әдёҖдёӘвҖҷcommonвҖҷж–Үд»¶еӨ№пјҢйҮҢйқўеҶҚеҲҶеҲ«е»әвҖҷfontsвҖҷе’ҢвҖҷstylusвҖҷж–Үд»¶еӨ№пјҢжҠҠи§ЈеҺӢдёӢжқҘзҡ„icomoonж–Үд»¶еӨ№йҮҢйқўfontsж–Үд»¶еӨ№дёӢзҡ„жүҖжңүж–Үд»¶жӢ·иҙқеҲ°vueйЎ№зӣ®йҮҢйқўзҡ„fontsж–Үд»¶еӨ№дёӢйқўпјҢз„¶еҗҺжҠҠ йҮҢйқўзҡ„style.cssжӢ·иҙқеҲ°vueйЎ№зӣ®йҮҢйқўзҡ„stylusдёӢйқўпјҢеӣ дёәжҲ‘иҝҷдёӘйЎ№зӣ®з”Ёзҡ„жҳҜstylusйў„еӨ„зҗҶеҷЁпјҢжүҖд»ҘжҲ‘жҠҠж–Үд»¶ж”№дёәдәҶstyle.styl
vueйЎ№зӣ®йҮҢйқўзҡ„и·Ҝеҫ„еҰӮеӣҫ
4гҖҒеңЁvueйЎ№зӣ®йҮҢйқўзҡ„main.jsйҮҢйқўеј•е…Ҙstyle.styl

import './common/stylus/index.styl'
